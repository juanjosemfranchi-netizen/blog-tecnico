Cómo automatizamos la creación de publicaciones para un e-commerce con IA

Por Juan José M. Franchi

Contexto del Proyecto

Automatización de creación de publicaciones para reducir el cuello de botella manual mediante OpenAI API.

Diagnóstico y Post-Mortem Técnico

Las pruebas iniciales mostraron fallos de consistencia. El problema era la falta de un contrato de instrucciones rígido en el prompt.

Acción Correctiva: Refactorización del prompt inyectando roles explícitos y estructura JSON fija.
Ver Commit de Refactorización del Prompt.

Lecciones Aprendidas

  • La IA requiere contextos parametrizados.
  • La automatización real requiere sanitización de datos.
  • Monitoreo: Implementamos un script de validación (ver tests.py) para asegurar la calidad del output JSON.

Reflexión sobre el Feedback

El feedback recibido permitió identificar que el proyecto era demasiado teórica. La aplicación de comunicación radicalmente sincera me llevó a implementar controles automáticos (tests.py) y mejorar la trazabilidad mediante una documentación rigurosa de cada commit.
